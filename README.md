# Handwriting and Audio AI Projects

This repository contains some basic AI projects I developed as a first approach to AI world, including neural networks for handwriting recognition and audio transcription using Whisper AI.

## Projects
### 1. Handwriting Recognition (MNIST)
#### 1.1 Convolutional Neural Network (CNN)

Description: A CNN model trained on the MNIST dataset to classify handwritten digits (0-9).

Features:

-Multiple convolutional layers

-Max-pooling layers

-Dropout for regularization

-Softmax output for classification

#### 1.2 Feedforward Neural Network (FNN)

Description: A fully connected neural network for the same MNIST classification task.

Features:

-Input flattened from 28x28 images

-Multiple dense layers with ReLU activation

-Output layer with softmax activation

### 2. Whisper AI (Audio Transcription)

Description: Audio transcription using OpenAI’s Whisper model.

Features:

-Supports multiple languages

-Converts speech to text from audio files

-Simple interface for batch processing

## Requirements

-Python 3.8+

-PyTorch

-openai-whisper
